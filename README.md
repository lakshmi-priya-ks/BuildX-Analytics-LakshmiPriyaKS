# IPL Data Analytics Project

## Dataset
IPL Dataset 2008–2025 — chosen for its rich player/team performance data and my prior experience analyzing it.

## Business Questions Answered
1. Which teams scored the most total runs across seasons?
2. Which players have the highest strike rates?
3. Which teams have the best bowling economy?

## Tools Used
Python (Pandas), SQL (SQLite), Power BI, GitHub

## Key Insight
Mumbai Indians scored the highest total runs across IPL seasons, while Virat Kohli emerged as the leading run-scorer and Yuzvendra Chahal recorded the highest number of wickets in the analyzed dataset.
